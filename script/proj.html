1.背景描述

  • 客户的问题是什么？（业务上的）
  • 客户希望将这个业务上的问题解决到什么程度？（量化指标）
  • 从数据挖掘的角度将这个问题解决到什么程度？（量化指标）
  • 达成项目目标的风险在哪里？
  • 对于关键的指标客户是如何定义的
  • 将业务问题转化成了哪些数据挖掘问题？ 

# read the data
df <- read.csv("../data/1_combinedData.csv")

2.数据理解

对于每一个指标
  • 缺失情况是什么样的？
  • 分布情况是什么样的？
对于重点有关系的指标
  • 指标与结果有没有显著关系
若不能通过，则需要思考
  • 现有数据是否能支撑业务的商业目标
  • 数据分析的目标是否正确
  • 解决问题的思路是否正确
